# Audio Files for Shakespeare Variorum

## All's Well That Ends Well

This directory should contain the audio file for "All's Well That Ends Well".

### Required File:
- `alls_well_that_ends_well.mp3`

### Where to Get Audio:

1. **LibriVox** (Free, Public Domain)
   - Visit: https://librivox.org/
   - Search for "All's Well That Ends Well"
   - Download the MP3 version
   - Rename to: `alls_well_that_ends_well.mp3`

2. **Internet Archive**
   - Visit: https://archive.org/
   - Search for "All's Well That Ends Well Shakespeare audiobook"

3. **YouTube to MP3**
   - Find a full reading on YouTube
   - Use a YouTube to MP3 converter
   - Save as `alls_well_that_ends_well.mp3`

### File Requirements:
- **Format**: MP3
- **Bitrate**: 128kbps or higher recommended
- **Filename**: Must be exactly `alls_well_that_ends_well.mp3`
- **Location**: Place in this directory

### Testing Without Audio:
The audio player will appear when "All's Well That Ends Well" is selected, but will show an error if the audio file is missing. All controls will still function once the file is added.

## Adding More Plays:

To add audio for other plays:
1. Add the MP3 file to this directory
2. Update `index.html` to include the play in the audio player logic
3. Update the audio source path in the HTML
